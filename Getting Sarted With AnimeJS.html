Lets start with a simple movement of a square.

In this example were going to take a simple square from the css file and make it move from left to right on the screen. Just like the one above!

First we'll import the library for anime.js by typing this inside of a script tag: scr="anime.min.js"

Then in a div tag insert the class for the square: class=".square"

Now inside another script tag were gonna put our code for the animation!

Insert this code into your script tag: anime({ targets: '.square', translateX: 250, loop: true });

You can use a loop to see it over again, or leave it out if you want the animation to only play on page load.
