What is anime.js?

Anime.js is a javascript library that specializes in animation! Its very simple, lightweight and is backed by a powerfull API and was created by gitlab user juliangarnier. It is the most popular and most used animation library today and works with CSS, DOM attributes, javascript objects and SVG. Its first public release was on June 27th 2016 and has had 3 versions of itself to date, the latest versoin being v3.2.1, which was released on October 12th. 2020. This library has been the fastest growing and has become the number 1 animation library on the open source market and for good reason.
It beats the basic animations that you can do with plain JS by along shot and it is easier to use!
There are other libraries that do very simliar things to anime.js such as velocity.js, which uses jquery style animation code. p5.js, Which has better documentation than anime.js but is more difficult to get your aniimations working. GSAP, which if very similiar to Anime.js but also has better documentation and a more robust API but needs special implementations for certain libraries to get them to work with it. However I found that Anime.js was the easiest and quickest to pick up and it being the most popular, I had to see what it offered.
Anime.JS is easy to set up too, just install the library in your folder or use the cdn. Theres also plenty of very good examples on how to use it. You can use it with basic webpages and css and it works with all modern browsers!
However, I found the documentation to be dissapointing in the fact that its not all there. Meaning that I would love some more thought and effort put into the documentation so that it be even easier to pick up!
Check out the GitHub here for more on Anime.JS

I find that if you want to add animation to your website, this is your best option! The only limit to this library is your own imagination! But for now lets move on to something really simple to get started with anime.js!


Lets create a simple animation of a square.

In this example were going to take a simple square from the css file and make it move from left to right on the screen. Just like the one above!

First we'll import the library for anime.js, if you've downloaded the library, by typing this inside of a script tag: 

src="anime.min.js "
If not then copy the url of this link!

Then in a div tag insert the class for the square: 

class=".square "

Now inside another script tag were gonna put our code for the animation!

Insert this code into your script tag: 

anime({
            targets: '.square',
            translateX: 500,
        });

You can use a loop to see it over again, or leave it out if you want the animation to only play on page load. 

loop: true

Heres what it looks like with a loop! 

class="squaretwo "
anime({
        targets: '.squaretwo',
        translateX: 500,
        delay: 3000,
        loop: true
    });

You can do a bunch of other stuff with targets too! Like rotating, skewing, direction, duration and morphing!

Here's an example of all of those put together!

 class="squarethree "
anime({
        targets: '.squarethree',
        borderRadius: ['0%', '50%'],
        translateX: 500,
        rotateX: 360,
        scale: 1,
        skew: 25,
        duration: 3000,
        delay: 1500,
        direction: 'alternate',
        loop: true
    });

You're now on your way to add animation to your own page! Good luck and happy coding! :)
